GoShopping_v3 - Flask + SQLite e-commerce sample (with categories, admin, and pagination)

Quick start:
1. Unzip and open terminal in GoShopping_v3 folder.
2. (Optional) create venv:
   python -m venv venv
   source venv/bin/activate   # mac/linux
   venv\Scripts\activate    # windows
3. Install dependencies:
   pip install flask pillow werkzeug
4. Run:
   python app.py
5. Open http://127.0.0.1:5000
